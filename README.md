# Amazon Dummy Homepage 🛒

A front-end practice project recreating the Amazon homepage layout — built to sharpen HTML & CSS skills using Flexbox and real-world UI structure.

## 📖 Overview

This is a static clone of Amazon's homepage, built purely for learning and portfolio purposes. It is **not affiliated with, endorsed by, or connected to Amazon.com, Inc.** — all branding is used only as a design reference for practicing front-end layout work.

Sections recreated:
- Top navbar — logo, delivery location, search bar with category dropdown, language selector, sign-in, returns & orders, and cart
- Secondary panel with "All" menu and quick links (Today's Deals, Gift Cards, Sell, Registry, Prime Video, Customer Service)
- Hero banner image
- A responsive grid of shopping category boxes (Clothes, Personal Care, Furniture, Electronics, Makeup, Pet Supplies, Toys, Fashion)
- Full multi-column footer with "Back to top," link columns, logo, and legal/copyright section

## 🛠️ Built With

- **HTML5** — semantic layout structure
- **CSS3** — Flexbox-based layout, hover states, and styling
- **[Font Awesome](https://fontawesome.com/)** (via CDN) — icons for location, search, and cart

No JavaScript or build tools are used — it's a static, framework-free page.

## 🚀 Getting Started

To run this project locally:

1. Clone the repository
   ```bash
   git clone https://github.com/CodexTalha/Amazon-Dummy-Hompage.git
   ```
2. Navigate into the project folder
   ```bash
   cd Amazon-Dummy-Hompage
   ```
3. Open `index.html` directly in your browser — no build step or dependencies required.
